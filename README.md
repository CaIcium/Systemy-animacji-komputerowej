# Systemy animacji komputerowej - LAB5

## Co zostało zrealizowane:
Na początek utworozono słownik z danymi na temat drzewa, krzewu i paproci. Następnie powstała funkcja stwórz rośline,
w której na początek tworzę pnie dla roślin, a następnie w zależności od typa pnia generuje liście. Kolejną funkcją było stworzenie systemu biomów,
czyli coś w stylu tranzcji między rodzajami lasu. Przykładowo w wnętrzu lasu jest dużo drzew naspenie naobrzezach jest mniej drzew, a więcej krzewów, a 
na samych końcach jest najwęcej paproci - małych roślin. Następnie została zrobina funkcja generacji lasu. Dodałem jeszcze podłoże
żeby lepiej się prezentowało.

## Uruchomienie:
Aby uruchomić skrypt trzeba pobrać plik skypt.py i w zakładce scripting w blenderze otworzyc plik tekstowy z rozserzeniem .py.
