# Systemy-animacji-komputerowej - LAB1
## Co zostało wykonane
W tym laboratorium były dwa zadania - Pierwsze gdzie trzeba było zrobić, żeby duszek odbijał się od ścan po wystrzale, a w drugim, trzeba było dodać 3 duszki i zrobić z tego skromny układ słoneczny gdzie ruszają się planety. gwiazdy.
Dodatkowo podzieliłem kluczowe fragmenty kodu w obu zadaniach na oddzielnie funkcje.
## Uruchomienie 
Aby uruchomić prgram wystarczy pobrać zadaniaA.sb3 albo zadaniaB.sb3 i na stronie Scratch załadować plik z dysku na stronę następnie kliknąć zieloną flage albo spacje.
