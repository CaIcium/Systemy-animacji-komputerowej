# Systemy-animacji-komputerowej - LAB06
## Co zostało zrealizowane 
Na początek dodałem podłoże, następnie kilka kolum i je powydłużałem za pomocą extrude, następnie jakby zrobiłem wejście do jaskini, dałem tam z kilka extrudów i kilka loop cutow. Pod koniec ustawiłem mniej więcej kamerę i dodałem troche nierówności podłodze.
## Uruchomienie
Wystarczy pobrac plik jaskinia.blnd i odpalić za pomocą blendera.
