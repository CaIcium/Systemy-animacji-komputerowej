# Systemy-animacji-komputerowej - LAB 11
# Co zostało wykonae 
W tym laboratorium trzeba bylo zanimować rośline wykonaną w labie 07 lub 06 używając pliku bpy. Na początek trzeba było zimportować rośline do nowego pliku. Następnie dla kazdego liścia znajdującego się na łodydze trzeba było dodać kołysanie się. Da kazego liścia trzeba było wykonać funkcjie def animuj_liść i pod koniec aniamcji czyścić animacjie funkcją def wyczyść_animacje().
Następnie trzeba było dać coś w stylu animacji otwierania pąka, funkcja odpowiadająca za aniacjie poką to def animuj_pąk(). Na końcu skryptu trzeba było jeszcze dodać głownego ifa który przygotowuje scene i uruchamia wszystkie funkcje.
# Uruchomienie
Aby uruchomić trzeba pobrać plik roslinaanimacjageometrii.py i roslinaozywiona11.blend. Utuchamiamy za pomocą blendera plik .blend i następnie w zakładce skryptowanie dąłoczamy z dysku plik .py i uruchamiamy skrypt.
