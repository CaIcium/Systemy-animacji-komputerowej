# Systemy-animacji-komputerowej - LAB3
## Co zostało wykonane 
W tym zadanaiu trzeba było wykońać animacje wystrzału fajerwerki a następnie ich wybuch, fajerwerki wystrzeliwowane są za pomocą przycisku myszki - tam gdzie klikniemy leci fajerwerka i następnie wybucha tworząc particles. Program jest podzielony na 3 funkcje Firework, FireworkShow i Particles. Kiedy wstrzelimy fajerwerkę wystarczająco blisko ziemi to wtedy widać ze particlesy spadają własnie na ziemie, a nie poza ekran. Złużyte fajerwerki i particlesy są usuwanie. Zadanie wykonałem bez automatycznego pokazu i suwaka.
## Uruchomeinie 
Tuaj mamy tylko jeden plik - index.html w którym jest też część skryptowa. Wystarczy pobrać i włączyć ten plik w przeglądarce.
